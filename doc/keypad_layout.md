Functional keys:
`NumLock` - unassigned
`/` - toggle NORMAL/INPUT mode
`*` - INPUT: enter word manually and add it to T9 dictionary
      NORMAL: toggle between T9 and manual mode
`-` - INPUT: cycle back suggestions
      NORMAL: undo
`+` - INPUT: cycle forward suggestions
      NORMAL: unassigned
`Enter` - unassigned
